# Task 21-22 — Feature Parity: Vegetation & Lighting Systems

## Summary

Implemented 6 feature parity items for the infinigen-r3f project:

### 1. Missing Deformed Tree Generators (3 files)

- **HollowTreeGenerator.ts** (~270 lines)
  - Generates hollow tree trunk with outer + inner cylinder
  - Annular caps (bottom and jagged top)
  - Knot holes and broken branch stubs
  - All geometry merged via `GeometryPipeline.mergeGeometries()`

- **RottenTreeGenerator.ts** (~370 lines)
  - Partially decomposed tree with missing bark, rot cavities, fungal growths
  - Broken branch stubs with snapped ends
  - Cavity displacement function for organic rot indentations

- **TruncatedTreeGenerator.ts** (~280 lines)
  - Tree stump cut at random height with ring-pattern UVs
  - u = radial distance, v = angular position → enables concentric ring rendering
  - Optional tilt/lean, flat or jagged cut

### 2. Cloud Type Variety

- **CloudTypes.ts** (~530 lines)
  - CumulusCloud, CumulonimbusCloud, StratocumulusCloud, AltocumulusCloud
  - Each with distinct geometry, altitude, density, opacity ranges
  - `getCloudLayerForType()` bridges to VolumetricClouds
  - `createCloudByType()` factory function

### 3. Blackbody Temperature Shader

- **BlackbodyShader.ts** (~280 lines)
  - `blackbodyToRGB()`, `blackbodyToHex()` — Tanner Helland algorithm (1000–40000K)
  - `createBlackbodyLight()`, `createBlackbodySpotLight()`, `createBlackbodyDirectionalLight()`
  - `updateLightTemperature()`, `getLuminousEfficacy()`
  - `BlackbodyLightPresets` and `COLOR_TEMPERATURES` constants

### 4. Procedural Lamp Geometry

- **ProceduralLamps.ts** (~420 lines)
  - `CeilingLampFactory`, `TableLampFactory`, `FloorLampFactory`
  - All use BlackbodyShader for correct color temperature
  - Configurable wattage (40–100W), temperature (2700–6500K), shade shape/color, base material
  - `LampFactoryResult` type with group, light, shade, bulb, config

### 5. Camera-Based Lighting

- Updated **LightingOrchestrator.ts**
  - `addCameraBasedLight(camera, intensity?)` → THREE.SpotLight
  - `removeCameraBasedLight()` → void
  - `updateCameraLight(camera)` → void (per-frame update)
  - `getCameraBasedLight()` → THREE.SpotLight | null

### 6. Updated Barrel Exports

- `atmosphere/index.ts` — CloudTypes exports
- `lighting/index.ts` — BlackbodyShader exports
- `trees/deformed-index.ts` — 3 new tree generators
- `vegetation/index.ts` — individual tree generators
